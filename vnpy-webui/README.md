# VnPy Web UI

Project: VnPy Web UI
Version: 1.0.0
Last Updated: 2026-02-28

---

## Project Overview

VnPy Web UI is a complete Web UI for VnPy quantitative trading platform.

---

## Features

1. Account Management
2. Position Management
3. Quote Display
4. Strategy Management
5. Backtesting
6. Trading
7. Data Management
8. Report Analysis

---

## Tech Stack

### Backend
- FastAPI 0.109.0
- Uvicorn 0.27.0
- SQLAlchemy 2.0.25
- Pydantic 2.5.0
- VnPy 3.9.0
- SQLite 3.45

### Frontend
- React 18.2
- TypeScript 5.3
- Ant Design 5.12
- Redux Toolkit 2.0
- React Router 6.22
- Axios 1.6
- Socket.IO Client 4.7

---

## Installation

### Backend

\`\`\`bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
\`\`\`

### Frontend

\`\`\`bash
cd frontend
npm install
npm start
\`\`\`

## Usage

### Backend

- URL: http://localhost:8000
- API Docs: http://localhost:8000/docs

### Frontend
